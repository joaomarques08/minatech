## 📖 Sobre o projeto

O **Minatech** é uma aplicação web criada para apresentar um programa de incentivo às mulheres na tecnologia.

O site reúne informações sobre a iniciativa, permitindo que as usuárias conheçam o projeto de forma simples, organizada e responsiva.

Além de divulgar o programa, o projeto busca proporcionar uma boa experiência de navegação através de uma interface agradável e acessível.

---

## ✨ Funcionalidades

- 🏠 Página inicial apresentando o programa
- 👩‍💻 Informações sobre a iniciativa

- ⚡ Navegação rápida e simples

## 🚀 Como executar

Clone o repositório:

```bash
git clone https://github.com/joaomarques08/minatech.git
```

Entre na pasta:

```bash
cd minatech
```
depois rode o programa com 

```bash
npm run dev
```

## 🎯 Objetivo

Este projeto foi desenvolvido como parte de um trabalho acadêmico, com foco na aplicação dos conhecimentos adquiridos durante o curso de desenvolvimento web.
